Requirements:

Python 3.4.3 - https://www.python.org/downloads/
Django compatible with Python 3.4.3 - https://www.djangoproject.com/
Numpy, Scipy, Scikit-learn, Sqlite packages for Python
Sqlite for the system

To install:

Simply unzip the folder named "Project"


To run:

1. Navigate to the directory Project/Main on a terminal window
2. Run "python manage.py runserver" to start the Django server
3. Navigate to the webpage using the IP that Django hosts the websites on with the suffix "/factors/home". For example: http://127.0.0.1:8000/factors/home/
4. Use the menu on the side to add filters for the restaurants considered for the analysis and visualizations. Each page has a description of what the visualization is about. Use the button on the main body of the page to run the analysis and display the graphs.
5. Use the top menu to navigate between other analyses and visualizations.
